# 家計簿 計算レギュレーション

## コストコ（オリコカード）の二重計上防止

コストコの支払いはオリコ提携カードを使用しており、月末に銀行口座から一括引落される。

**ルール:**
- コストコのレシートを受け取ったら → 品目別に明細に入力する
- 銀行口座の「自払 オリコ」の引落 → **明細に記録しない**（レシートで計上済みのため）

**コストコ品目の費目分類:**

| 品目 | 費目 |
|------|------|
| 食品・飲料（酒類以外） | 食費 |
| お酒類 | 健太郎お小遣い |
| 日用品・洗剤・消耗品 | 日用品 |
| ガソリン | 自動車 |
| その他 | 内容に応じて判断 |

---

## 外貨積立（ITA）の記録ルール

- 毎月銀行から引き落とされた**円額をそのまま**「外貨積立」として明細に記録する
- 為替レートにより月ごとに金額が変動するため、固定予算と実績に差が出るのは正常
- 予算プランの¥20,000はあくまで参考値
